# C2 Profile

Tariq uses C2 Profiles to automated some of the agent behaviours. You can do the following:
  • Sleep Time (DNS and HTTP only)
  • Add customizable HTTP headers for both server and client
  • Setting IP, Port, ListenerType, and PayloadType
